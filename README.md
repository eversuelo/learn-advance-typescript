# TypeScript Algorithms & Software Design

A structured learning path covering TypeScript fundamentals, clean code practices, software design principles, and engineering methodologies.

---

## 🗺️ Learning Path

| # | Topic | Description |
|---|-------|-------------|
| 01 | [Introduction](./01-introduction/) | TypeScript basics: types, functions, classes, generics |
| 02 | [Clean Code](./02-clean-code/) | Naming, DRY, functions, modules, classes |
| 03 | [SOLID](./03-solid/) | The five SOLID design principles with examples |
| 04 | [Design Patterns](./04-design-patterns/) | Creational, Structural, and Behavioral patterns |
| 05 | [TDD](./05-tdd/) | Test-Driven Development: Red → Green → Refactor |
| 06 | [DDD](./06-ddd/) | Domain-Driven Design: entities, aggregates, bounded contexts |
| 07 | [Data Structures](./07-data-structures/) | Algorithms and complexity analysis |
| 08 | [Software Engineering](./08-software-engineering/) | Agile, Scrum, Kanban, XP |

---

## 🛠️ Running Examples

```bash
# Run any TypeScript file directly
npx ts-node <path-to-file>.ts
```
