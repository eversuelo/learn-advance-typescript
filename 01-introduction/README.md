# Introduction to TypeScript

A progressive tour through TypeScript's core features.

## 📚 Topics

| # | Topic | Description |
|---|-------|-------------|
| 01 | Data Types | Primitives, arrays, tuples, enums |
| 02 | Functions & Objects | Typed functions, optional params, object shapes |
| 03 | Interfaces & Types | `interface` vs `type`, extending, merging |
| 04 | Classes & OOP | Access modifiers, inheritance, abstract classes |
| 05 | Generics | Reusable typed components |
| 06 | Advanced Types | Union, intersection, conditional types |
| 07 | Modules & Namespaces | `import`/`export`, module resolution |
| 08 | Decorators | Metadata, class/method decorators |
| 09 | Utility Types | `Partial`, `Pick`, `Omit`, `Record`… |
| 10 | Async Patterns | Promises, async/await, error handling |
| 11 | Type Guards | `typeof`, `instanceof`, custom guards |
| 12 | Conditional Types | `infer`, distributive types |
| 13 | Mapped Types | `keyof`, `in`, template literal types |
| 14 | Best Practices | `strict` mode, avoid `any`, common pitfalls |
